### M1-2 `contracts/bom.py`

| 类 | 字段 | 类型 | 说明 |
|---|---|---|---|
| `BOM` | `clause` | str | 条款名 |
| | `block_code` | str | 编码（唯一） |

| | `semantic_definition` | str | 定义（自然语言） |
| | `extraction_rules` | ExtractionRules | 抽取规则 |

| | `core_match_rules` | List[ExtractionRule] | 匹配规则 |
| `RecallProfile` | `positive_keywords` | List[str] | 正向关键词（短词、程序过滤） |
| | `confusion_words` | List[str] | 易混淆词 |
| | `section_hints` | List[str] | 章节提示 |
| | `semantic_queries` | List[str] | 语义查询句 |

## M4 · anonymize（1 个文件，杨力，独立）

### `nodes/anonymize.py`

| 函数 | 输入 | 输出 | 说明 |
|---|---|---|---|
| `anonymize(text)` | str | `Tuple[str, dict]` | 脱敏文本 + 映射表 |

- 公司名 → `【采购方】/【供应商】/【公司_1】`（按上下文角色）
- 金额 → `【金额_1】`
- 人名 → `【人名_1】`
- 手机号 → `【手机号】`（掩码）
- 证件号 → `【证件号】`（保留首3尾4）
- 联系方式（邮箱/地址）→ `【联系方式】`

## M6 · generate（1 个文件，林大宇，依赖 M1/M3/M5）

### `nodes/generate.py`

| 函数 | 输入 | 输出 | 说明 |
|---|---|---|---|
| `generate(test_set, clause_name)` | TestSet, str | `BOM` | Stage1+2+3 |
| `optimize(current_bom, badcases, trace?)` | BOM, list, Optional[StructuredTrace] | `BOM` | 诊断→修订→自检 |
| `_stage1_definition(data)` | dict | dict | 调 LLM 出定义+规则 |
| `_stage2_recall(s1_result, data)` | dict, dict | dict | 调 LLM 出画像 |
| `_stage3_verify(bom, positives, negatives)` | BOM, list, list | `Verification` | 调 LLM 自检 |

> `generate` = 无 badcase，直接生成。`optimize` = 有 badcase + 当前 BOM，先诊断再修订。
